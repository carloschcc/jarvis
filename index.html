Instalação Rápida em XAMPP

1. Preparar XAMPP
• Baixar arquivos para C:\xampp\htdocs\ad-manager\
• Habilitar extension=ldap no php.ini
• Reiniciar Apache no painel de controle
2. Primeiro Acesso
• Acessar: http://localhost/ad-manager
• Login inicial: admin / admin123
• Sistema carrega automaticamente
3. Configurar LDAP
• Ir em "Configurações" no menu
• Inserir dados servidor AD: IP, porta, domínio, Base DN
• Testar conexão e salvar configurações
4. Gerenciar Usuários
• Acessar "Usuários" para ver lista completa
• Buscar, ativar/bloquear, resetar senhas
• Sistema totalmente funcional!
