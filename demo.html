Como Instalar em Produção

1. Preparar Ambiente
  • Instalar XAMPP no Windows
  • Habilitar extensão LDAP no PHP
  • Configurar Apache virtual host
  • Reiniciar serviços
2. Instalar Sistema
  • Extrair arquivos para htdocs
  • Configurar permissões da pasta
  • Acessar via navegador
  • Login: admin / admin123
3. Configurar LDAP
  • Ir em "Configurações"
  • Inserir dados do servidor AD
  • Testar conexão LDAP
  • Salvar configurações
